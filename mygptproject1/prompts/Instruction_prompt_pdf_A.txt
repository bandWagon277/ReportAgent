### 4.1 Standard Report Structure
SRTR reports follow a consistent pipeline:
1. **Study Population Description** - Cohort definition, inclusion/exclusion criteria counts, baseline demographics
2. **Baseline Characteristics Table** - Demographics stratified by comparison groups (e.g., dialysis vs. waitlist vs. transplant recipients)
3. **Primary Outcome Analysis** - Rates, hazard ratios, or interrupted time-series results
4. **Subgroup Analysis** - Outcomes stratified by age, race, sex, disease cause, donor quality
5. **Trend/Policy Analysis** - Longitudinal trends, pre/post comparisons if applicable

**Kaplan-Meier survival curves** (REQUIRED when time-to-event data is available):
Use the `lifelines` library to produce publication-quality KM curves:

```python
from lifelines import KaplanMeierFitter
from lifelines.statistics import logrank_test

kmf = KaplanMeierFitter()
fig, ax = plt.subplots(figsize=(10, 7))

# Stratify by groups — collect median survival per group
groups = df['sex'].dropna().unique()
median_survivals = {}
for grp in groups:
    mask = df['sex'] == grp
    kmf.fit(
        durations=df.loc[mask, 'follow_up_days'] / 30.4375,  # convert to months
        event_observed=df.loc[mask, 'death_event'],
        label=grp
    )
    kmf.plot_survival_function(ax=ax, ci_show=True)
    # IMPORTANT: kmf.median_survival_time_ is a SCALAR (float), not a dict!
    # Store it per group in your own dict:
    med = kmf.median_survival_time_
    median_survivals[grp] = float(med) if not np.isinf(med) else None

ax.set_xlabel('Time (months)', fontsize=12)
ax.set_ylabel('Survival Probability', fontsize=12)
ax.set_title('Kaplan-Meier Survival Estimates by Sex', fontsize=14)
ax.legend(frameon=False, fontsize=11)
ax.set_ylim(0, 1.05)
ax.grid(axis='y', alpha=0.3)

# Log-rank test between first two groups
grp_list = list(groups)
if len(grp_list) >= 2:
    m1, m2 = df['sex'] == grp_list[0], df['sex'] == grp_list[1]
    lr = logrank_test(
        df.loc[m1, 'follow_up_days'], df.loc[m2, 'follow_up_days'],
        df.loc[m1, 'death_event'], df.loc[m2, 'death_event']
    )
    log_rank_p = round(lr.p_value, 4)

# data_summary for this figure — use median_survivals dict, NOT kmf.median_survival_time_.get()
# "data_summary": {
#     "chart_type": "kaplan_meier",
#     "key_stats": {"median_survivals": median_survivals, "log_rank_p_value": log_rank_p},
#     "data_points": [...subset or summary, NOT full raw rows...],
#     "insights": [...]
# }
```
